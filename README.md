# 🏨 StayEase Hotel Management System

## Project Overview

StayEase Hotel Management System is a Java-based application developed to simplify and automate hotel management operations. The system helps manage customer details, room bookings, check-in/check-out processes, billing, and room availability efficiently.

I created this project to learn Java programming and database connectivity by developing a hotel management system. This project helped me understand how hotel management systems work and how Java can be used to manage customer and booking records.

---

## Features

* Add, update, and manage customer details
* Room booking 
* Check-in and check-out system
* Track room availability
* Generate customer bills automatically
* Simple and user-friendly interface
* Database connectivity for storing records

## Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/VAISHNVi148/StayEase-Hotel-Management.git
```

### 2️. Open the Project

Open the project in your preferred Java IDE.

### 3️. Configure Database

* Create a database
* Update database credentials in the project
* Create required tables if needed

### 4️. Run the Project

```bash
Run Main.java
```

## How It Works

1. User/Admin logs into the system
2. Add customer details
3. Book rooms based on availability
4. Manage check-in and check-out
5. Generate customer bill automatically
6. Store records in the database
